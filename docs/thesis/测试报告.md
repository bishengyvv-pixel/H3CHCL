## 3. 非功能性测试

### 3.1 容错测试

| 测试场景 | 预期行为 | 实际结果 |
|----------|----------|----------|
| 单台设备 IP 不可达 | 跳过该设备，继续评估其他设备 | ✓ 跳过，日志输出失败原因 |
| 设备认证失败 | 标记 ERROR，不再重试 | ✓ 标记并跳过 |
| 命令执行失败 | 该命令输出为空字符串，不中断 | ✓ 空字符串参与 regex 匹配（视为不通过） |
| 无设备配置 | 退出并提示 | ✓ 输出 "无设备配置，退出。" |
| 损坏的 YAML 文件 | 抛出明确错误 | ✓ YAML 解析异常直接报出 |

|----------|------|------|

### 4.2 发现的系统问题

| 问题 | 严重程度 | 备注 |
|------|----------|------|
| 日志审计(info-center)未在任一台设备开启 | 高 | 建议所有设备开启 |
| DHCP Snooping 在 6 台接入交换机上均未配置 | 高 | 接入层必配 |
| BPDU 保护未在接入层开启 | 高 | 防 STP 攻击 |
| 核心层路由协议未配置 MD5 认证 | 高 | 核心层必配 |
| 管理平面 ACL 缺失 | 中 | 建议配置 |
| NTP 时间同步未配置 | 低 | 建议配置 |

> 注：此处的"问题"是检测到的**目标设备安全缺陷**，非系统 Bug。
